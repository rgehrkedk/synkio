# Synkio Import Plugin

Figma plugin for importing JSON design tokens as Variables.

## Features

- Upload JSON files (drag & drop or browse)
- Configure collections and modes
- Preset for three-tier token systems
- Works with any JSON token format

## Installation

This plugin will be published to Figma Community.

For development:

1. Run `npm run build`
2. In Figma: Plugins → Development → Import plugin from manifest
3. Select `manifest.json`

## Usage

1. Open Figma file
2. Run "Synkio Import"
3. Upload JSON token files
4. Configure collections
5. Click "Import"
