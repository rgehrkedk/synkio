# Synkio Dashboard

Web dashboard for previewing and syncing Figma design tokens.

## Features

- Preview tokens visually (colors, typography, spacing)
- Export to multiple formats (CSS, Tailwind, JSON)
- Sync wizard with step-by-step guidance
- No authentication required for preview

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deployed to Vercel at [synkio.io](https://synkio.io)
